# 🎮 Rust - Guessing Game 🎯

## 💡 What is this?

🎲 **Rust - Guessing Game** is a terminal-based thriller where your job is simple:  
Guess the **secret number** between 1 and 100.  
But beware — the machine doesn't go easy on anyone. 😈
